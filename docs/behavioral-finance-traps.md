# 投资心理陷阱：别让情绪偷走你的收益

行为金融学揭示了投资者常见的非理性偏差，即使拥有完美的选股策略，若无法克服心理弱点，收益也会大打折扣。

## 常见认知偏差
### 1. 损失厌恶
- **表现**：亏损1000元的痛苦远大于盈利1000元的快乐，导致过早卖出盈利股、死扛亏损股。
- **对策**：提前设定止盈止损规则，机械执行，不盯盘。

### 2. 锚定效应
- **表现**：过度依赖某个价格（如成本价或前期高点），例如“等回本就卖”，结果可能越套越深。
- **对策**：以客观估值和策略信号为依据，成本价只是历史数据。

### 3. 确认偏误
- **表现**：只寻找支持自己判断的信息，忽视相反证据。
- **对策**：主动阅读反对观点，加入讨论群时不要只看“唱多”的声音。

### 4. 过度自信
- **表现**：短期成功后夸大自身能力，频繁交易，导致成本升高、胜率下降。
- **对策**：记录投资日志，定期回顾，用数据评价自己的决策质量。

### 5. 羊群效应
- **表现**：看到别人都在买，自己也跟风追逐热门股，忽视估值。
- **对策**：依赖策略选股（如本工具的低估值和质量成长策略），用规则代替感觉。

## 如何训练投资纪律？
- **使用模拟盘**：在真实资金入场前，先用工具选股做模拟组合，培养耐心。
- **设定投资日记**：每次交易记录买入逻辑，复盘时检查是否遵守策略。
- **定期再平衡**：每季度或每半年调整仓位，恢复原始配置，避免情绪化决策。
- **认知到“错过”是常态**：没有人能抓住每一次机会，只赚能力圈内的钱。
