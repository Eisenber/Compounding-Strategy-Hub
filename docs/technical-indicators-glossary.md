# 常用技术指标名词解释

技术指标是通过对价格、成交量等数据进行数学运算得出的数值，用于辅助判断趋势、动能、超买超卖状态。

## 趋势类指标

### MA（移动平均线）
- **SMA（简单移动平均）**：过去 N 个周期收盘价的算术平均数。5日线、20日线、60日线等。
- **EMA（指数移动平均）**：给予近期价格更高权重，更灵敏。12日 EMA 和 26日 EMA 常用于 MACD。
- **用途**：判断趋势方向；均线交叉产生买卖信号。

### MACD（指数平滑异同移动平均线）
- **构成**：由快线 DIF（12日 EMA - 26日 EMA）、慢线 DEA（DIF 的 9日 EMA）、柱状线（DIF - DEA）组成。
- **信号**：
  - DIF 上穿 DEA，柱状线由负转正（金叉），买入信号。
  - DIF 下穿 DEA，柱状线由正转负（死叉），卖出信号。
  - 顶背离（价格新高，DIF 未新高）→ 趋势可能反转下跌。
  - 底背离（价格新低，DIF 未新低）→ 可能见底反弹。

## 摆动类指标

### RSI（相对强弱指数）
- **计算**：一段时间内上涨幅度之和 ÷ (上涨幅度+下跌幅度之和) × 100。常见参数为 14日。
- **区间**：0-100。通常 RSI > 70 为超买，价格可能回调；RSI < 30 为超卖，价格可能反弹。
- **应用**：结合趋势使用，在强趋势中 RSI 可长期停留在超买区，不宜单纯依据 RSI 反向操作。

### KDJ（随机指标）
- **构成**：K 线（快速确认线）、D 线（慢速主干线）、J 线（方向敏感线）。
- **信号**：K、D 值在 20 以下为超卖，80 以上为超买；J 值超过 100 或低于 0 为极端。
- **金叉/死叉**：K 线上穿 D 线为买进信号，下穿为卖出信号。
- **注意**：KDJ 对盘整市有效，但在单边趋势中易钝化。

### WR（威廉指标）
- **计算**：反映收盘价在过去 N 日价格区间中的相对位置。
- **区间**：0-100（或 100-0）。高于 80 超卖，低于 20 超买。
- **用法**：通常与 RSI 互补，确认超买超卖信号。

## 量价类指标

### OBV（能量潮）
- **逻辑**：价格变动需成交量支撑。当日收盘价高于前一日，将当日成交量累加；低于前一日，则减去成交量。
- **信号**：OBV 线方向应与价格趋势一致，若价格创新高而 OBV 未同步新高，是趋势可能衰竭的警告。

### 成交量比率（VR）
- **计算**：N 日内上涨日成交量之和 ÷ N 日内下跌日成交量之和。
- **区间**：VR > 350 警惕超买，VR < 40 密切关注超卖反弹。

## 压力支撑类指标

### 布林带（Bollinger Bands）
- **构成**：中轨为 20日 SMA，上轨为中轨 + 2倍标准差，下轨为中轨 - 2倍标准差。
- **信号**：
  - 价格触及上轨可能回调，触及下轨可能反弹。
  - 带宽收缩表示波动率降低，可能酝酿变盘；带宽扩张表示趋势加速。
  - 价格沿上轨运行属强势；沿下轨运行属弱势。

### 斐波那契回调线
- **用法**：在一段趋势的高低点间画出 23.6%、38.2%、50%、61.8% 等位置作为潜在的支撑/阻力。
- **常用参数**：61.8% 和 38.2% 最为关键。

## 指标结合使用原则
- 不要单独依赖一个指标，最好趋势指标（如 MA、MACD）与摆动指标（如 RSI）结合。
- 在震荡市中，摆动指标准确率高；在趋势市中，趋势指标更可靠。
- 必须结合成交量验证信号。

*本名词解释仅为教学目的，不构成买卖建议。*
