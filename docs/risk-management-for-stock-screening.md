# 选股中的风险管理：你必须避开的坑

任何选股策略都需要配套风险过滤，否则筛选结果可能包含大量“价值陷阱”或“财务暴雷”股。

## 系统内置风险标签
当筛选出某只股票时，我们的策略引擎会自动生成风险提示，例如：
- **高负债率**：资产负债率超过 60% 则标注。
- **盈利波动**：如果近两年净利润增速标准差过大，提示不稳定。
- **行业周期**：强周期行业（如钢铁、化工）会提示行业波动风险。
- **ST 预警**：直接排除 ST 或 *ST 股票。
- **停牌风险**：排除当前停牌的公司。

## 个人应增加的过滤习惯
1. **审计意见**：避免使用被出具“保留意见”或“无法表示意见”财报的公司。
2. **现金流检查**：经营活动现金流净额应长期为正，否则利润可能只是纸面数字。
3. **关联交易**：大额异常关联交易可能涉及利益输送。
4. **商誉占比**：商誉超过净资产 30% 的公司需警惕减值风险。

## 仓位管理
即便策略选出的股票，也建议：
- 单一股票仓位不超过总资金的 5%
- 单一行业仓位不超过 30%
- 永远保留部分现金，应对极端行情
