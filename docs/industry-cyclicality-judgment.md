# 行业景气度判断：如何识别赛道冷热

行业景气度决定了大部分股票的β（系统性风险），即使公司自身优秀，处于下行周期时股价也容易承压。

## 行业生命周期
1. **初创期**：技术未成熟，市场认知低，风险极高（如早期的光伏、基因编辑）。
2. **成长期**：营收高速增长，竞争者大量涌入，龙头初现（如当前的部分AI应用领域）。
3. **成熟期**：增速放缓，格局稳定，利润丰厚（如白酒、白色家电）。
4. **衰退期**：需求萎缩，产能过剩，被迫转型（如传统燃油车配件、胶卷）。

## 宏观经济与行业轮动
- **复苏期**：可选消费、科技、金融通常领先。
- **过热期**：原材料、能源、工业表现突出。
- **滞胀期**：防御性行业（医药、公用事业）相对抗跌。
- **衰退期**：消费必需品、高股息资产更受追捧。

## 如何量化观察行业景气度？
- **PMI（采购经理人指数）**：>50 表示制造业扩张，利好工业、原材料。
- **行业产销数据**：如汽车销量、房地产新开工面积、半导体出货量等。
- **产业政策**：国家的减税、补贴、规划会直接催生行业景气。
- **价格指数**：产品价格上涨（如锂、硅料）会增加行业利润。

## 用数据看景气度
在你使用的选股工具中，`stocks` 表包含 `industry` 字段，你可以：
- 统计某个行业的股票平均 ROE 变化趋势（如果同行业多数公司 ROE 下滑，可能景气见顶）。
- 观察营收增长率的中位数，判断行业扩张速度。
- 比较市盈率分位：行业中位数 PE 处于历史高位时，需警惕泡沫。

## 风险提示
- 景气高峰时估值也高，戴维斯双杀风险大。
- 政策变动（如“双减”对教育行业）可能导致景气骤停。
- 普通投资者不需要预测拐点，等待右侧确认（例如行业指数站上60日均线）再参与更安全。
