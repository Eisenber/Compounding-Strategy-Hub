# 基金定投策略：用纪律战胜波动

定投（定期定额投资）是本工具“复利对比”功能的完美实践对象，它通过分批买入来摊平成本，降低择时风险。

## 定投为什么有效？
- **微笑曲线定律**：当市场先跌后涨时，定投的成本随之下移，反弹后更快盈利。
- **克服人性弱点**：避免追涨杀跌，强制储蓄，用时间换取复利。
- **无需预判市场**：只要最终方向向上，过程中的波动反而是朋友。

## 适用标的
- **宽基指数基金**：如沪深300、中证500、标普500，长期趋势向上，波动适中。
- **优质行业指数**：消费、医药、科技等长期增长赛道，但需注意行业周期。
- **高股息策略基金**：与我们的“高股息策略”筛选逻辑一致，定期分红可再投入。

## 关键参数设定
- **定投频率**：每月一次最普遍，与工资流入匹配；周定投可平滑但交易频繁。
- **定投金额**：根据可支配收入，建议用闲钱，避免影响日常生活。
- **止盈不止损**：设定目标收益率（如年化10%或总收益30%），达到后分批赎回，市场下跌时不恐慌卖出，反而加码买入低价份额。

## 进阶策略：价值平均定投
- 每月调整投入金额，使账户市值按计划增长（例如每月增加1000元市值）。市场下跌时需投入更多，上涨时投入减少，甚至赎回超额部分。这种方式更贴近“低买高卖”。

## 与本工具的联动
1. 打开“复利对比”功能，创建一个方案模拟宽基指数基金定投（年化收益率参考历史长期回报7%-9%）。
2. 再创建一个方案模拟银行存款（收益率2%左右），对比长期差距。
3. 利用选股策略，寻找符合“高股息”或“质量成长”的股票构成的指数基金，手动组合你的定投标的。
4. 跟踪数据新鲜度，定投日可选择在每月发薪日附近，心理负担小。

## 风险提示
- 历史收益不代表未来表现，指数可能长期低迷（如日本1990-2010）。
- 定投亏损时不要中断，否则会错失低位买入的机会。
- 基金费率会侵蚀复利，优先选择管理费低的产品（如指数基金）。
