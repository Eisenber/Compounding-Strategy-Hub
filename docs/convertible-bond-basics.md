# 可转债基础：下有保底，上不封顶？

可转债（Convertible Bond）是上市公司发行的、可以按约定价格转换为公司股票的债券。它具有“债性”保底和“股性”进攻的双重特征。

## 核心要素
- **面值**：通常为 100 元，发行价及到期赎回价的基础。
- **票面利率**：一般较低（0.3%-3%），逐年递增，远低于普通债券。
- **转股价**：约定将债券转换为股票时的价格，如转股价 10 元，每张转债可换 100/10=10 股。
- **转股价值** = 正股价 ÷ 转股价 × 100，代表转债内含的股票价值。
- **转股溢价率** = (转债市价 - 转股价值) ÷ 转股价值，越低越接近“股性”，负溢价存在套利空间。
- **强赎条款**（提前赎回权）：如果正股价连续一段时间超过转股价 130%，公司有权赎回，迫使投资者转股。这是转债的“天花板”。
- **回售条款**：若股价持续低迷，触发条件时投资者有权将转债按 100+利息卖回给公司，这是“保底”的关键。
- **下修条款**：正股价低于转股价一定比例时，公司可下调转股价，修复转股价值，利好转债持有者。

## 收益与风险特征
- **防守性**：只要公司不违约，最差情况拿回本金+少量利息（依赖回售或到期赎回）。
- **进攻性**：当正股大涨时，转债价格会跟随上涨，收益率接近股票。
- **主要风险**：
  - 信用风险：上市公司资不抵债，可转债违约（虽极少，但存在）。
  - 高价买入风险：若转债价格远高于 130 元且溢价率高，安全垫消失，波动加大。
  - 强制赎回风险：忘记在强赎截止前转股或卖出，将以低价被强制赎回，导致损失。

## 简易投资策略
1. **双低策略**：价格 ≤ 120 元 + 转股溢价率 ≤ 20%，兼顾债性保护与股性弹性。
2. **打新策略**：参与转债申购，上市首日卖出，历史上破发概率较低。
3. **下修博弈**：关注正股价格持续低于转股价、财务状况好的公司转债，潜伏下修预期。
4. **到期正收益策略**：买入价格接近面值且到期收益率接近定期存款的数年剩余时间转债，持有到期。
