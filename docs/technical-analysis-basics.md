# 技术分析基础：读懂价格与成交量的语言

技术分析通过研究市场交易数据（价格、成交量、时间）来预测价格趋势，它不关心公司基本面，而是关注“市场行为包含一切信息”。

## 三大核心假设
1. **市场行为包容消化一切**：所有已知信息都已反映在股价中。
2. **价格以趋势方式演变**：一旦形成趋势，价格更可能继续原有方向。
3. **历史会重演**：人性不变，曾经有效的价格形态往往会重现。

## 关键分析工具

### 1. K线图
- **阳线**：收盘价高于开盘价，通常为红色或白色。
- **阴线**：收盘价低于开盘价，通常为绿色或黑色。
- **上下影线**：代表当日最高价与最低价，反映多空分歧程度。

### 2. 趋势与均线
- **移动平均线（MA）**：平滑价格曲线，识别方向。常用 5日、20日、60日、120日均线。
- **多头排列**：短期均线在长期均线之上，呈上升趋势。
- **空头排列**：短期均线在长期均线之下，呈下降趋势。
- **金叉/死叉**：短期均线上穿长期均线为金叉（看涨）；下穿为死叉（看跌）。

### 3. 成交量
- **价升量增**：上涨健康，趋势可靠。
- **价升量缩**：上涨乏力，警惕回调。
- **天量天价**：巨量之后往往出现阶段性顶部。

### 4. 常用指标
- **MACD**：利用短期与长期指数平滑移动平均线的差离值，判断趋势转折。
- **RSI（相对强弱指标）**：衡量价格变动速度，>70 超买，<30 超卖。
- **布林带**：中轨为20日均线，上下轨为中轨±2倍标准差，价格触及上下轨时常有反转可能。

## 技术分析的局限性
- 无法预测突发事件（政策、黑天鹅）。
- 在小盘股上容易被资金操纵。
- 需要结合基本面过滤，避免纯图形陷阱。
- 胜率并非100%，必须配置止损规则。

## 与本选股工具的结合
- 我们的选股策略选出的是基本面优秀的股票，你可以进一步用技术工具寻找合适的买入点位（如：股票池中，价格回调至60日均线且成交量萎缩时）。
- 技术分析不是选股逻辑，而是择时工具，两者互补。
