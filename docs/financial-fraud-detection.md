# 财务造假识别：避开报表里的地雷

财务造假是投资者面临的最大非系统性风险之一。通过分析财务报表间的勾稽关系与异常信号，可以大幅降低踩雷概率。

## 常见的造假动机
- 维持上市地位（避免 ST 或退市）
- 推高股价以便大股东减持或质押融资
- 满足融资条件（如增发、发债）
- 管理层业绩对赌压力

## 关键识别指标

### 1. 净利润与经营现金流长期背离
- 若连续数年净利润增长，但经营活动现金流净额持续为负或远低于净利润，说明利润可能只是“纸面数字”，并未真正收到现金。
- **参考阈值**：经营现金流净额 ÷ 净利润 < 0.7 应警惕，< 0.3 高度危险。

### 2. 应收账款增速远高于营收增速
- 放宽信用政策、虚增合同可推高收入，但会沉淀为应收账款。
- **公式**：应收账款增速 / 营收增速 > 1.5 时，存在提前确认收入或虚构交易的可能。

### 3. 存货异常增长
- 存货积压可能意味着产品滞销，也可能隐藏成本（少结转成本以虚增利润）。
- 若存货增速超过营收增速，且存货周转天数明显变长，需审计其跌价准备是否充足。

### 4. 毛利率异常波动
- 毛利率与行业均值偏差过大，或毛利率在新产品/新区域突然飙升，但无合理技术或成本优势解释，可能是虚增收入或少计成本。

### 5. 商誉暴雷风险
- 商誉占净资产比例超过 30% 被视为高商誉风险。
- 并购标的业绩承诺期满后利润骤降，往往是大额商誉减值的前兆。

### 6. 关联交易与资金占用
- 向关联方销售占比过高（>30%）需关注定价是否公允。
- 其他应收款中存在大额非经营性资金往来，大股东可能占用资金。

### 7. 审计意见
- 标准无保留意见 ≠ 一定安全，但非标准意见（保留意见、否定意见、无法表示意见）基本意味着直接排除。

## 快速排雷检查清单
- [ ] 经营现金流/净利润 > 0.7
- [ ] 应收增速 ≤ 营收增速 + 20%
- [ ] 存货增速不超过营收增速两倍
- [ ] 商誉/净资产 < 30%
- [ ] 审计意见为标准无保留
- [ ] 最近三年无证监会立案调查或问询函
