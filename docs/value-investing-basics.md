# 价值投资基础：寻找被低估的好公司

价值投资的核心思想是：**市场短期是投票机，长期是称重机**。股价最终会向企业的内在价值回归。

## 核心指标
- **市盈率 (PE)**：股价 ÷ 每股收益。反映市场愿意为每 1 元盈利支付的价格。PE 越低，回本理论年限越短。
- **市净率 (PB)**：股价 ÷ 每股净资产。对银行、保险等重资产行业尤其重要，PB < 1 意味着跌破净资产。
- **股息率**：每股分红 ÷ 股价。高股息率往往意味着公司现金流稳健，且估值可能偏低。

## 本策略模板：低估值策略
我们的“低估值”模板组合了多个条件，避免单一指标陷阱：
- PE(TTM) ≤ 15 ：盈利能力合理
- PB ≤ 1.8 ：资产估值偏低
- ROE ≥ 8% ：证明企业具备持续盈利的基础
- 利润增长 ≥ 0 ：拒绝业绩持续下滑的公司

## 为什么不能只看 PE？
- 周期股在利润高点时 PE 极低，但随后利润可能暴跌（“低 PE 陷阱”）。
- 净利润含非经常性损益时 PE 失真。
- 高杠杆企业看似 PE 低，但风险敞口大。

## 适用人群
适合希望长期持有、相信均值回归的投资者。
