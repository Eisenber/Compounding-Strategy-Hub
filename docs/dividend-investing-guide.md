# 股息投资：用分红构建现金流护城河

股息投资即寻找**长期稳定分红、股息率较高**的上市公司，类似“收租”逻辑。

## 核心指标
- **股息率**：年度分红总额 ÷ 当前股价。长期高于 3% 就有吸引力。
- **分红率**：每股分红 ÷ 每股收益。超过 80% 可能不可持续。
- **派息历史**：连续 5 年以上逐年提高分红的公司，管理层回报股东的意愿更明确。

## 本策略模板：高股息策略
筛选条件：
- 股息率 ≥ 4% ：现金回报可观
- ROE ≥ 8% ：盈利足以支撑分红
- 负债率 ≤ 70% ：避免高杠杆下的虚假高分红

## 警惕“红利陷阱”
- 股价暴跌导致股息率被动升高（公司基本面可能已恶化）。
- 一次性特别分红让股息率失真。
- 周期性行业在利润高峰期分红慷慨，低谷期可能断息。

## 优势
- 现金流相对确定，适合退休规划或厌恶风险的投资者。
- 复利再投可以大幅提升长期收益。

## 建议
将股息再投资到同一策略中，利用“红利再投”发挥复利威力，这正是本工具“复利对比”功能的最佳实践场景。
